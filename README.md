# ✨ Zoe的40天奇妙旅程

一个为 Zoe 定制的纯静态网页游戏。40 个探索站点，对应 40 天成长计划。

> 从加拿大带回的行囊，装满独特的经历和视野。
> 现在，一起探索中国的课堂，让两个世界在你心中相遇。
> —— 爸爸妈妈

## 使用方式

1. 在平板浏览器打开 `index.html`（或部署后的 GitHub Pages 网址）
2. 每天完成一个探索站：闪卡复习 → 视频课程 → 在线练习 → 线下任务
3. 星光和足迹自动保存在浏览器中

## 部署（GitHub Pages）

1. 在 GitHub 创建仓库（如 `zoe-journey`），上传本文件夹
2. Settings → Pages → Source: main → Save
3. 获得网址：`https://你的用户名.github.io/zoe-journey/`

## 技术说明

- 零框架，纯 HTML/CSS/JS
- localStorage 存储进度
- Web Speech API 朗读古诗
- 打印样式支持线下练习

## 内容来源

基于人教版小学 4-6 年级语数教材，配套国家中小学智慧教育平台和 B 站公开课。
